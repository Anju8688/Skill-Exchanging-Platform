# Skill Exchange Platform

## Description
This project is a web application where users can share and learn different skills. Users can register, add their skills, and view skills shared by others.

## Features
- User registration and login
- Add skills
- View other users' skills
- Basic skill exchange system

## Technologies Used
- Python (Flask)
- HTML, CSS
- PostgreSQL

## How to Run

1. Install required packages:
   pip install -r requirements.txt

2. Create database:
   python init_db.py

3. Run the application:
   python run.py

4. Open in browser:
   http://127.0.0.1:5000/

## Project Structure
- run.py: Main file to start the application
- config.py: Configuration and database settings
- init_db.py: Used to create database tables
- requirements.txt: List of required packages
